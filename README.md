# Lógica de Programación - Ejercicio 1 

Este proyecto consiste en un programa en JavaScript que solicita tres números al usuario, analiza sus valores y los muestra ordenados de mayor a menor y de menor a mayor. Además, identifica si los números ingresados son iguales.

---

##  Descripción del Problema

El programa realiza los siguientes pasos:
1. Solicita al usuario **3 números** mediante un `prompt()` y los almacena en sus respectivas variables.
2. Convierte los datos ingresados a tipo `number` y valida que los valores sean numéricos para evitar errores de ejecución.
3. Analiza los datos para identificar cuál es el número **mayor**, el **del centro** y el **menor**.
4. Imprime por la consola (o en el DOM) los números ordenados:
   - **De mayor a menor**
   - **De menor a mayor**
5. Detecta si todos los números son iguales e imprime un mensaje en caso afirmativo.

---

##  Casos de Prueba

El programa fue verificado utilizando las siguientes entradas y salidas de prueba:

* **Entrada:** `4, 4, 2`
  * **Salida de mayor a menor:** `4, 4, 2`
  * **Salida de menor a mayor:** `2, 4, 4`

* **Entrada:** `4, 2, 4`
  * **Salida de mayor a menor:** `4, 4, 2`
  * **Salida de menor a mayor:** `2, 4, 4`

* **Entrada:** `2, 4, 4`
  * **Salida de mayor a menor:** `4, 4, 2`
  * **Salida de menor a mayor:** `2, 4, 4`

---
